Data Science: Python oder R?

Zuletzt aktualisiert am Nov 26, 2020

Wenn es nach Kaggle-Nutzer*innen geht: Welche Programmiersprache ist führend im Bereich Data Sciene wenn es um die Häufigkeit der Nutzung und der Empfehlungen geht?

2018 wurde die Kaggle Nutzergruppe nach ihren bevorzugten Data Science Methoden und Werkzeugen sowie ihre Erfahrungen damit befragt. Unglaubliche 23860 Nutzer*innen weltweit haben geantwortet. The folgenden Analysen basieren auf dem öffentlich verfügbaren Datensatz:

Python oder R? Diese Frage hat das Potential, in lebhafte, kontroversielle Diskussionen auszuarten, wie z.B. auf medium oder quora zu sehen ist. Oder kommt vielleicht noch eine dritte Programmiersprache ins Spiel? Kaggler*innen zumindest geben eine ziemlich eindeutige Antwort…

Python wird am häufigsten benutzt

Gefragt nach der einen, am häufigsten benutzten Programmiersprache, antworten 15222 der Befragten, nämlich 53.7%, mit Python.

8637 von 23860 haben diese Frage nicht beantwortet.

Python wird auch von Nutzer*innen anderer Programmiersprachen empfohlen

Sogar Teilnehmer*innen, die Python nicht als primäre Sprache verwenden, empfiehlt ein großer Anteil Data Science Aspirant*innen Python als die erste Sprache, die es zu erlernen gilt.

Wurde eine dieser zwei Fragen nicht beantwortet, zählt das als “None”. Teilnehme*innen, die beide Fragen nicht beantwortet haben, wurden nicht gezählt (n=5052). Insgesamt wurden also 18807 Antworten gezählt.

Mehrheit empfiehlt Python als erste Programmiersprache

Python wird von 75.48% der 18788 Teilnehmer*innen als erste Sprache, die es beim Einstieg in das Thema Data Science zu erlernen gilt, empfohlen.

5071 Teilnehmer*innen habe keine Antwort gegeben.

Python zunehmend populär

Pythons Popularität hat zugenommen, wie ein Blick auf die Erfahrung der Teilnehmer*innen im Bereich Data Science zeigt. Während “nur” 39.18% der Teilnehmer*innen mit einer Erfahrung im Bereich Data Science von 30-40 Jahren Python als am häufigsten verwendete Sprache angeben, erhöht sich dieser Wert auf 59.89% bei den Teilnehmer*innen mit einer Erfahrung von 1-2 Jahren. Neueinsteiger*innen (<1 Jahr Erfahrung oder noch keine Programmiererfahrung im Bereich Datenanalyse) mit Programmiererfahrung aus anderen Bereichen weisen eine diversere Palette an häufig verwendeten Sprachen auf - aber immer noch mit Python als Vorreiter.

*5326 Teilnehmer*innen haben nicht geantwortet. 41 Teilnehmer*innen die noch nie Code geschrieben haben und es auch nicht lernen wollen (have never written code and do not want to learn) wurden auch nicht gezählt.
